# 📌 Projeto DevNotes

> Aplicação mobile desenvolvida em React Native para gerenciamento de anotações, com foco em aprendizado de estrutura, navegação e manipulação de estado.

---

## 🧠 Contexto

Este projeto foi desenvolvido no início dos meus estudos em React Native, com base em um curso prático, onde implementei e adaptei funcionalidades para reforçar meu aprendizado.

O objetivo foi compreender, na prática, o funcionamento do ecossistema mobile com React Native, incluindo desde o setup do ambiente até a construção de funcionalidades completas.

> ⚠️ Projeto de estudo baseado em curso, com adaptações e melhorias próprias.

---

## 🚀 Tecnologias utilizadas

- React Native
- Node.js
- JavaScript (ES6+)

---

## 📦 Como executar o projeto

### Pré-requisitos

- Node.js instalado
- Expo CLI ou ambiente React Native configurado
- Emulador Android/iOS ou dispositivo físico

### Executando

```bash
# Clone o repositório
git clone <url-do-repo>

# Acesse a pasta do projeto
cd Projeto_DevNotes

# Instale as dependências
npm install

# Execute o projeto
npm start
